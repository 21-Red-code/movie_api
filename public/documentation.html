Movie API

This API will return a list of movies to a user based on their request and gets more detailed description about a certain movie, director or genre based on user's request.

Request URL Method Request data format Response data format
Request a list of ALL movies movies GET None Returns a JSON object with all the movies from the data base
Request data with detail description about a single movie by title /movies/:Title GET None Returns a JSON object with all the information about the specified movie
Request data about a genre /genres/:Name GET None Returns a JSON object with a description about the specified genre
Request data about a director /directors/:Name GET None Returns a JSON object with data about a director
Add a new user /users/ POST A JSON object with a new user name, email,password and birthdate A JSON object with a status lines of code about the submission
Login to a user acoount /login POST A JSON object with a user name and password A JSON object with a user profile info and a token
Updates a user info /users/:Username PUT A JSON object with the user new informtion A JSON object with a status line of code about the submission
Adds the selected movie in a user FavMovies-list /users/:Username/:MovieID PUT A JSON object with the selected movie title A JSON object with a status line of code about the submission
Removes a movie from a user FavMovies-list /users/:Username/FavMovies/:MovieID PUT None A JSON object with a status line of code about the submission
Removes a user account from the APP /users/:Username DELETE None A message with a status line about the submission
